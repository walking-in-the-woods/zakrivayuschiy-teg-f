# Проект «Закрывающий тег»

Верстка и публикация статичного сайта.

## Технологии

- HTML5, CSS3
- Адаптивная вёрстка
- Вариативные шрифты
- CSS-анимации и фильтры
- Модальное окно через `<dialog>`

## Особенности

- Фиксированный градиентный фон
- Анимированная иконка сердца
- Кнопки с эффектом заполнения через `mix-blend-mode`
- 8 карточек с личными текстами и CSS-фильтрами

## Запуск

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере
